# FYP

iOS app built for my final year project. Helps autistic children aged 5-10 complete daily routines in a calm way.

## How to run it
1. Open CalmStepsApp.xcodeproj in Xcode
2. In InsightEngine.swift swap YOUR_KEY_HERE for a real OpenAI API key
3. Add your own GoogleService-Info.plist for Firebase to work
4. Run on any iPhone simulator

## Why some files are missing
- GoogleService-Info.plist is excluded for security -- AVAILABLE IN XCODE ZIP FILE PROVIDED
- OpenAI API key is a placeholder -- AVAILABLE IN XCODE IN THE ZIP FOLDER PROVIDED IN SUBMISSION
- The app works fine without both just use guest mode on the onboarding screen
